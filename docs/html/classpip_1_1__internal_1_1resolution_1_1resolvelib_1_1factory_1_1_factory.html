Iterator[Requirement] pip._internal.resolution.resolvelib.factory.Factory._make_requirements_from_install_req ( self,
InstallRequirement ireq,
Iterable[str] requested_extras )
protected 
Returns requirement objects associated with the given InstallRequirement. In
most cases this will be a single object but the following special cases exist:
    - the InstallRequirement has markers that do not apply -> result is empty
    - the InstallRequirement has both a constraint (or link) and extras
        -> result is split in two requirement objects: one with the constraint
        (or link) and one with the extra. This allows centralized constraint
        handling for the base, resulting in fewer candidate rejections.
