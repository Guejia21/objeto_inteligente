\doxysubsection{Descripción detallada}
\begin{DoxyVerb}Filename globbing utility. Mostly a copy of `glob` from Python 3.5.

Changes include:
 * `yield from` and PEP3102 `*` removed.
 * Hidden files are not ignored.
\end{DoxyVerb}
