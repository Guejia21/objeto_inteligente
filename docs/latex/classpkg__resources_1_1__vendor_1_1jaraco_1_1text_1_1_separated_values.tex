\doxysubsection{Descripción detallada}
\begin{DoxyVerb}A string separated by a separator. Overrides __iter__ for getting
the values.

>>> list(SeparatedValues('a,b,c'))
['a', 'b', 'c']

Whitespace is stripped and empty values are discarded.

>>> list(SeparatedValues(' a,   b   , c,  '))
['a', 'b', 'c']
\end{DoxyVerb}
