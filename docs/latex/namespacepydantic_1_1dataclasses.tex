! abstract "{}\+Usage Documentation"{} {\ttfamily dataclasses}

A decorator used to create a Pydantic-\/enhanced dataclass, similar to the standard Python {\ttfamily \doxylink{namespacepydantic_1_1dataclasses_a3e4ce12843ff856477810ee31a564004}{dataclass}}, but with added validation.

This function should be used similarly to {\ttfamily \doxylink{namespacepydantic_1_1dataclasses_a3e4ce12843ff856477810ee31a564004}{dataclasses.\+dataclass}}.

Args\+: \+\_\+cls\+: The target {\ttfamily \doxylink{namespacepydantic_1_1dataclasses_a3e4ce12843ff856477810ee31a564004}{dataclass}}. init\+: Included for signature compatibility with {\ttfamily \doxylink{namespacepydantic_1_1dataclasses_a3e4ce12843ff856477810ee31a564004}{dataclasses.\+dataclass}}, and is passed through to {\ttfamily \doxylink{namespacepydantic_1_1dataclasses_a3e4ce12843ff856477810ee31a564004}{dataclasses.\+dataclass}} when appropriate. If specified, must be set to {\ttfamily False}, as pydantic inserts its own {\ttfamily \+\_\+\+\_\+init\+\_\+\+\_\+} function. repr\+: A boolean indicating whether to include the field in the {\ttfamily \+\_\+\+\_\+repr\+\_\+\+\_\+} output. eq\+: Determines if a {\ttfamily \+\_\+\+\_\+eq\+\_\+\+\_\+} method should be generated for the class. order\+: Determines if comparison magic methods should be generated, such as {\ttfamily \+\_\+\+\_\+lt\+\_\+\+\_\+}, but not {\ttfamily \+\_\+\+\_\+eq\+\_\+\+\_\+}. unsafe\+\_\+hash\+: Determines if a {\ttfamily \+\_\+\+\_\+hash\+\_\+\+\_\+} method should be included in the class, as in {\ttfamily \doxylink{namespacepydantic_1_1dataclasses_a3e4ce12843ff856477810ee31a564004}{dataclasses.\+dataclass}}. frozen\+: Determines if the generated class should be a \textquotesingle{}frozen\textquotesingle{} {\ttfamily \doxylink{namespacepydantic_1_1dataclasses_a3e4ce12843ff856477810ee31a564004}{dataclass}}, which does not allow its attributes to be modified after it has been initialized. If not set, the value from the provided {\ttfamily \doxylink{namespacepydantic_1_1config}{config}} argument will be used (and will default to {\ttfamily False} otherwise). config\+: The Pydantic config to use for the {\ttfamily \doxylink{namespacepydantic_1_1dataclasses_a3e4ce12843ff856477810ee31a564004}{dataclass}}. validate\+\_\+on\+\_\+init\+: A deprecated parameter included for backwards compatibility; in V2, all Pydantic dataclasses are validated on init. kw\+\_\+only\+: Determines if {\ttfamily \+\_\+\+\_\+init\+\_\+\+\_\+} method parameters must be specified by keyword only. Returns\+: A decorator that accepts a class as its argument and returns a Pydantic {\ttfamily \doxylink{namespacepydantic_1_1dataclasses_a3e4ce12843ff856477810ee31a564004}{dataclass}}.

\begin{DoxyVerb}Try to rebuild the pydantic-core schema for the dataclass.

This may be necessary when one of the annotations is a ForwardRef which could not be resolved during
the initial attempt to build the schema, and automatic rebuilding fails.

This is analogous to `BaseModel.model_rebuild`.

Args:
    cls: The class to rebuild the pydantic-core schema for.
    force: Whether to force the rebuilding of the schema, defaults to `False`.
    raise_errors: Whether to raise errors, defaults to `True`.
    _parent_namespace_depth: The depth level of the parent namespace, defaults to 2.
    _types_namespace: The types namespace, defaults to `None`.

Returns:
    Returns `None` if the schema is already "complete" and rebuilding was not required.
    If rebuilding _was_ required, returns `True` if rebuilding was successful, otherwise `False`.
\end{DoxyVerb}
