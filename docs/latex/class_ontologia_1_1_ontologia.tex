\doxysubsection{Descripción detallada}
\begin{DoxyVerb}Gestor de operaciones sobre la ontología perteneciente al objeto.
Proporciona carga, consultas (SPARQL), inserciones, actualizaciones y
operaciones entre ontologías (resta). Cada método realiza la
persistencia correspondiente en el fichero RDF asociado.\end{DoxyVerb}
