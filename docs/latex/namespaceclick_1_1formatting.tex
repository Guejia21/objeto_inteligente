 str click.\+formatting.\+wrap\+\_\+text (\begin{DoxyParamCaption}\item[{str}]{text}{, }\item[{int }]{width}{ = {\ttfamily 78}, }\item[{str }]{initial\+\_\+indent}{ = {\ttfamily "{}"{}}, }\item[{str }]{subsequent\+\_\+indent}{ = {\ttfamily "{}"{}}, }\item[{bool }]{preserve\+\_\+paragraphs}{ = {\ttfamily False}}\end{DoxyParamCaption})}

\begin{DoxyVerb}A helper function that intelligently wraps text.  By default, it
assumes that it operates on a single paragraph of text but if the
`preserve_paragraphs` parameter is provided it will intelligently
handle paragraphs (defined by two empty lines).

If paragraphs are handled, a paragraph can be prefixed with an empty
line containing the ``\\b`` character (``\\x08``) to indicate that
no rewrapping should happen in that block.

:param text: the text that should be rewrapped.
:param width: the maximum width for the text.
:param initial_indent: the initial indent that should be placed on the
                       first line as a string.
:param subsequent_indent: the indent string that should be placed on
                          each consecutive line.
:param preserve_paragraphs: if this flag is set then the wrapping will
                            intelligently handle paragraphs.
\end{DoxyVerb}
