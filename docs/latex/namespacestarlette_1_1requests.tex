 dict\mbox{[}str, str\mbox{]} starlette.\+requests.\+cookie\+\_\+parser (\begin{DoxyParamCaption}\item[{str}]{cookie\+\_\+string}{}\end{DoxyParamCaption})}

\begin{DoxyVerb}This function parses a ``Cookie`` HTTP header into a dict of key/value pairs.

It attempts to mimic browser cookie parsing behavior: browsers and web servers
frequently disregard the spec (RFC 6265) when setting and reading cookies,
so we attempt to suit the common scenarios here.

This function has been adapted from Django 3.1.0.
Note: we are explicitly _NOT_ using `SimpleCookie.load` because it is based
on an outdated spec and will fail on lots of input we want to support
\end{DoxyVerb}
