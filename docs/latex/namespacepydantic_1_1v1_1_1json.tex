 Union\mbox{[}int, float\mbox{]} pydantic.\+v1.\+json.\+decimal\+\_\+encoder (\begin{DoxyParamCaption}\item[{Decimal}]{dec\+\_\+value}{}\end{DoxyParamCaption})}

\begin{DoxyVerb}Encodes a Decimal as int of there's no exponent, otherwise float

This is useful when we use ConstrainedDecimal to represent Numeric(x,0)
where a integer (but not int typed) is used. Encoding this as a float
results in failed round-tripping between encode and parse.
Our Id type is a prime example of this.

>>> decimal_encoder(Decimal("1.0"))
1.0

>>> decimal_encoder(Decimal("1"))
1
\end{DoxyVerb}
